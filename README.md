# Weather_App-using-fetch-API-and-DOM-Java-Script-
This project is a responsive weather forecasting web application built using HTML, CSS, and JavaScript. 
The application allows users to enter the name of a city and view real-time weather information fetched from a public weather API.

The project is built by using HTML,CSS,JS{
*DOM
**Fetch API
*function
*event listeners  etc.,}


Features 

**you can even search the current   Weather of the your village or Town near by.**

1.Shows the current Time and Date on the screen.
2.Changes the backgrount according to the Temperature of the city we searched.
3.Displays:  
  *Temperature
  *Feels Like Temperature
  *Weather Condition
  *Humidity
  *Wind Speed
  *Pressure
  *Visibility
  *Sunrise Time
  *Sunset Time
  *UV Index
4.Search weather by city name.
5.Real-time weather data using API.
6.Modern UI with weather icons.
